Color Encoding Learning

From Basic Vision to Advanced Color Models

1. What is Color?

Color is a perceptual property of light, not an inherent feature of objects. Objects do not have color themselves—they only reflect or absorb specific wavelengths of visible light.

Key rule: Objects reflect the light we see; they absorb the light we don’t see.

2. Visible Light Spectrum

Visible light is a small part of the electromagnetic spectrum, ranging from 400 nm (violet) to 700 nm (red). Different wavelengths correspond to different hues.

Visible light spectrum and wavelength distribution

3. Three Core Properties of Color

  • Hue: The dominant color name (red, green, blue), determined by the main wavelength of light.
  • Saturation: Color purity. High saturation = no white light mixed; low saturation = grayish.
  • Brightness: Subjective perceived light intensity from a surface.

4. Brightness vs Luminance

Brightness: Subjective human perception, no strict math definition.

Luminance: Objective physical value, calculated based on wavelength and light power.

Human eyes are more sensitive to green light, so same-power lights have different perceived brightness.

1. Human Eye Structure for Color Vision

The retina contains two photoreceptor cell types: rods and cones.

Retina structure showing rods and cones distribution

  • Rods: Work in low light, no color detection.
  • Cones: Work in bright light, responsible for color vision.

2. Three Types of Cones

  • L cones (Red): Peak sensitivity ~610 nm
  • M cones (Green): Peak sensitivity ~560 nm
  • S cones (Blue): Peak sensitivity ~430 nm

Color blindness is usually caused by missing or defective cone types.

3. Tristimulus Theory

All perceived colors can be reproduced by mixing red, green, blue light in proper proportions. This is the foundation of all digital color systems.

4. Metamers

Metamers are different light spectra that produce identical color perception in human eyes. Different wavelength combinations can look like the same color.

RGB Color Model

RGB is an additive color model used for screens (monitors, phones, TVs). Colors are created by emitting red, green, blue light.

Additive Principle

More light = brighter color. (0,0,0) = black; (255,255,255) = white

RGB additive color mixing diagram

Common Values

  • Red: (255, 0, 0)
  • Green: (0, 255, 0)
  • Blue: (0, 0, 255)
  • Gray: (128, 128, 128)

YUV / YCbCr Color Model

YUV separates luminance (Y) and chrominance (U/V). YCbCr is the digital version for image/video compression (JPEG, MPEG).

Why Separate Luma and Chroma?

Human eyes are much more sensitive to brightness than color. We can reduce chroma data to save space without visible quality loss.

Y = brightness; Cb = blue difference; Cr = red difference

CMYK Color Model

CMYK is a subtractive model for printing. Inks absorb light instead of emitting it.

Subtractive Principle

C(cyan), M(magenta), Y(yellow) are subtractive primaries. K = black (Key color).

CMY cannot make pure black because real inks have impurities. So K (black ink) is added.

HSV Color Model

HSV is human-intuitive color space for designers: Hue, Saturation, Value.

  • Hue (0-360°): Color angle (0°=red, 120°=green, 240°=blue)
  • Saturation (0-100%): Color purity
  • Value (0-100%): Brightness
