# Prototype

This folder contains a static front-end prototype for the first assessment.

## Files
- `index.html`: main prototype entry
- `styles.css`: visual style and responsive layout
- `app.js`: mocked data and simple front-end interactions

## How to use
Open `index.html` in a browser. No build step and no backend are required.

## Prototype coverage
- TA profile and CV upload
- TA job browsing and application submission
- TA application status visibility
- MO job posting management
- MO applicant review and decision updates
- Admin workload monitoring

## Notes
- This is a static prototype only.
- Data is mocked in the browser for demonstration.
- The final coursework implementation can later replace the mocked state with Java Servlet/JSP pages and local JSON file storage.
