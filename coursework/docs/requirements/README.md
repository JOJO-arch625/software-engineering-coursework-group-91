# Requirements Documents

- `TA-Recruitment-System-SRS.en.md`: English SRS aligned section-by-section with the Chinese draft for coursework use.

Recommended usage order:
1. Review and revise the Chinese draft with the team.
2. Confirm scope, roles, sprint plan, and acceptance criteria.
3. Use the confirmed content to fill the product backlog template and build the prototype.
4. Keep both files aligned when requirements change.
